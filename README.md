# Hyprlan

**Hyprlan** is a dedicated hackathon community designed to empower students by helping them learn, build, and actively participate in hackathons. Our mission is to foster innovation, collaboration, and practical skill development among the next generation of creators.

## Overview

We believe that hackathons are one of the best environments for students to gain hands-on engineering experience, meet like-minded peers, and build meaningful projects. Hyprlan provides the platform and community resources to make this journey accessible, educational, and engaging for everyone.

### What We Offer
- **A Supportive Community**: A welcoming environment where students can collaborate, form teams, and learn from one another.
- **Skill Development**: Opportunities to enhance technical, design, and problem-solving skills in a fast-paced setting.
- **Exciting Events**: Streamlined access to join and compete in our premier 48-hour hackathons.

---

## Technical Stack

The Hyprlan platform is built with a modern, high-performance tech stack ensuring a smooth experience for all participants:

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Database/ORM**: Prisma & Drizzle
- **Authentication**: NextAuth.js
- **API**: tRPC

## Running Locally

To run the platform on your local machine:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to view the application.

# Hyprlan

Hyprlan is a stunning, high-performance landing page for a 48-hour competitive hackathon. Designed with modern aesthetics inspired by tech and developer conferences like Devcon.

## Technologies Used
- Next.js 15
- Tailwind CSS
- Prisma & Drizzle ORM
- NextAuth.js
- tRPC

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
